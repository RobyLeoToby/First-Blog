Let's discuss about Django

Django is a Python framework for building websites. It is relatively easy to use and it helps you to finish your site faster. This framework uses the Model-View-Controller software pattern, meaning that you can develop your site in a logical way
  • Model - the central component of the pattern responsible for managing the data of the application. It receives user input from the controller.
  • View - any representation of information, meaning the representation of the model in a particular way.
  • Controller - accepts input and converts it to commands for the model or the view.
